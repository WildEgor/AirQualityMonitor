# AirQualityMonitor

A smart air quality monitor based on ESP32 and CCS811 sensor

## Features

1. Measures eCO₂ and TVOC levels  
2. Publishes the data to MQTT topics  
3. Displays CO₂ levels on a screen and via an RGB LED
